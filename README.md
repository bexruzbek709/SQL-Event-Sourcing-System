# 🚀 SQL Event-Sourcing & Data Versioning System

Ushbu loyiha ma'lumotlar bazasi arxitekturasida ma'lumotlar tarixini saqlash va har bir o'zgarishni audit qilishni avtomatlashtirishga bag'ishlangan.

### 🔥 Asosiy imkoniyatlar:
*   **Temporal Tables:** SQL Serverning tizimli versiyalash funksiyasi orqali ma'lumotlarning istalgan vaqtdagi holatini ko'rish.
*   **JSON Audit Logging:** Ma'lumotlar o'zgarganda (UPDATE/DELETE) eski holatni JSON formatida arxivlash.
*   **Time-Travel Querying:** Maxsus yaratilgan `Stored Procedure` yordamida o'tmishdagi ma'lumotlarni qayta tiklash.

### 🛠 Texnologiyalar:
- T-SQL (Microsoft SQL Server)
- SSMS (SQL Server Management Studio)
